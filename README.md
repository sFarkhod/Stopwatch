# Stopwatch

A simple multi-stopwatch web app built with React, Vite, and Tailwind CSS.

## Features

- Add multiple independent stopwatches
- Start, pause, resume, and reset each stopwatch
- Delete any stopwatch
- Responsive and modern UI

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Run the app locally:**
	```bash
	npm run dev
	```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/Stopwatches.tsx` – Manages the list of stopwatches
- `src/Stopwatch.tsx` – Stopwatch component logic and UI
- `src/components/Button.tsx` – Reusable button component

## Tech Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
